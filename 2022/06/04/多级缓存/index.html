本地进程缓存

分布式缓存

  • 优点:存储容量更大，可靠性更好，可以在集群间共享
  • 缺点:访问缓存有网络开销
  • 场景:缓存数据量较大，可靠性要求较高，需要在集群间共享

进程本地缓存

  • 优点:读取本地内存，没有网络开销，速度更快
  • 缺点:存储容量有限，可靠性较低，无法共享
  • 场景:性能要求较高，缓存数据量较小

获取请求参数

参数格式 参数实例 参数解析代码示例
路径占位符 /item/1001 1.正则表达式匹配location ~ /item/(\d+)
2.匹配到的参数会存入到ngx.var数组中,可以用角标获取
local id = ngx.var[1]
请求头 id:1001 local headers=ngx.req.get_headers()
local id = headers.id
GET请求参数 ?id=1001 local getParams=ngx.req.get_uri_args()
local id = getParams.id
POST表单参数 id=1001 ngx.req.read_body()
local postParams = ngx.req.get_post_args()
local id = postParams.id
JSON参数 {“id”:1001} ngx.req.read_body()
local jsonBody = ngx.req.get_body_data()
local id = jsonBody.id
